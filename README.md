# Eldomaine Secondary School Website

A modern, user-friendly school website designed to make important information accessible to learners, parents, and staff.  

The platform showcases upcoming events, admission information, school resources, and announcements in a clear and student-friendly format.

Built with **React** and **Vite** for fast development and optimized production builds.
